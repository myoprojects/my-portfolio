Vision-Based BBTV Detection System

A full-stack, AI-powered system that detects Banana Bunchy Top Virus using computer vision. Designed to support early disease detection for farmers through a low-cost, portable solution.

Project Preview

About the Project

This project focuses on detecting Banana Bunchy Top Virus (BBTV) through image-based and real-time camera analysis. The system uses a YOLOv5 deep learning model trained on banana leaf images and is deployed on a Raspberry Pi for portability.

A Flask backend handles image processing and inference, while a responsive web interface allows users to upload images or perform real-time detection. Detection results and logs are stored in a MySQL database.

Results & Impact

The system achieved high accuracy in controlled and real-world environments, demonstrating reliable BBTV detection in real time. It provides an accessible tool for farmers and agricultural workers to identify infected plants early and reduce crop loss.
