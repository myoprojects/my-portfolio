Vision-Based BBTV Detection

An AI-powered system for detecting Banana Bunchy Top Virus using YOLOv5, Raspberry Pi, and a full-stack web platform.

The Problem

Banana Bunchy Top Virus is difficult to detect early, leading to crop loss and delayed intervention for farmers.

The Solution

A portable vision-based detection system that classifies banana leaves as Healthy or BBTV using deep learning and real-time image processing.
